터미널 기능

터미널에서 가능한 기능들이 적혀있습니다.

이 위키는 Zeekerss 의 인디 협동 공포 게임 Lethal Company 의 비공식 정보창입니다!



명령어들






SPECIAL CODE

팀원의 모니터에서 보이는 특수코드를 입력시
에어락을 열고 닫을 수 있으며 터렛 및 각종 함정을 끌 수 있습니다. (예 : E9)









BESTIARY

괴물 및 생명 스캔 시 정보가 수집되며
Bestiary [괴물이름] 또는 INFO [괴물이름]을 검색하여 정보를 볼 수 있다.









BUY

Buy [아이템 이름] [갯수]시 아이템을 살 수 있으며
[아이템 이름] 3 을 입력시 해당 아이템을 3개 살 수 있다



SHOP

Shop 입력시 아이템 이름과 가격을 보여주며
날마다 세일할 수 있으며 세일하는 템은 랜덤이며 세일률도 랜덤이다. (단위 : %)



아이템 구매 시 날마다 세일할 수 있으며 세일하는 템은
랜덤이며 세일률도 랜덤이다. (단위 : %)

또한 INFO [아이템이름]으로 해당 아이템의 설명을 읽어볼 수 있다.









CONFIRM, CON, C

특정 명령어를 입력 후 확인을 하기 위해 쓰는 명령어이다.



DENY, DEN, D

특정 명령어를 입력 후 취소를 하기 위해 쓰는 명령어이다.



EJECT

자신을 포함한 모든 승무원을 퇴출시키려면 해당 명령어를 쓴다. 달 궤도 주위에 돌고 있어야한다.









HELP

터미널 명령어들이 무엇이 있는지 보는 명령어이다.



MOONS

갈 수 있는 위성들의 목록을 볼 수 있다.
해당 위성들의 날씨를 볼 수 있으며
위성 이름 입력시 해당 위성을 가는데 비용을 나타내준다.
INFO[위성이름]으로도 확인할 수 있으며 [위성이름]만 쳐서 이동할 수 있다.









SCAN

현재 위성에 남아있는 폐품의 총 갯수와 총 몇 $인지 표시해주는 명령어이다.









SIGURD

위성들에서 확인할 수 있는 SIGURD의 기록들을 얻고 그 기록들을 읽어볼 수 있는 명령어이다.









STORAGE

B키를 눌러 저장한 사물들이 보관되어 있는 명령어입니다.
명령어를 치고 [사물 이름]을 치면 다시 꺼낼 수 있다.









SWITCH [직원 이름]

메인모니터에서 해당 직원의 화면을 볼 수 있게 해준다.
또한 VIEW MONITOR 명령어로 메인 모니터를 ON / OFF 할 수 있다.









레이더 부스터

PING [레이더 부스터 이름] : 해당 레이더 부스터에서 소음이 나게 한다.
FLASH [레이더 부스터 이름] : 해당 레이더 부스터에서 괴물이 기절하는 섬광이 나오게 한다.
